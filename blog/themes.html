Dark Side

My eyesight is terrible, and staring at a computer screen all day definitely doesn't help my sensitive eyes. Because of this, I am a huge believer in dark themes for just about everything I do on the computer. Whether it is the environment I'm currently working in (in this case I'm using the Atom text editor with, you guessed it, the One Dark theme enabled), or watching YouTube videos, you can bet I will be using a dark theme if it's available. So here are some of the themes I implement in my everyday life to keep my eyes happy

Note: these are my opinions and what works for me, so definitely play around to find what works for you!

Firefox CSS

One of my favorite features of using Firefox is that users can completely redesign the UI using CSS. There is even a dedicated subreddit for users to post their own code and get feedback. My CSS is primarily take from /u/Guerra24's post. I changed some colors to better suit my personal taste and added some extra features that were not included in his file. My code can be found here. If you don't know how to set up your own CSS on Firefox, follow this guide.

RStudio Theme

I love RStudio. It's objectively the best IDE for R, and I don't think anything else even comes close to it. I work in it everyday for at least a couple of hours, so naturally I want to be able to look at my screen for a while without destroying my eyes and getting headaches. The photo above is my theme for RStudio. Here is my setup:

  • RStudio Theme: Modern
  • Editor Font: Courier
  • Editor Font Size: 12
  • Editor Theme: Tomorrow Night Bright (although sometimes I'll use Tomorrow Night cause it's a little lighter!)

Juptyer Notebook

I don't typically use Jupyter Notebooks for Python, but they definitely have a place in my programming environment arsenal. There is this super great repo that allows you to change your JN theme. Installation instructions are found in the README! After a successful installation, copy and paste this in your terminal:
jt -t oceans16 -f roboto -fs 12
